Aula 3 - Formulários, CSS e Armazenamento Local

Objetivo da Aula:
    Nesta aula, colocamos em prática conceitos fundamentais de desenvolvimento web. O objetivo foi criar um formulário funcional, estilizá-lo com CSS (revisando seletores, cores, espaçamentos e responsividade) e implementar a lógica em JavaScript para armazenar os dados no localStorage do navegador.


Conteúdo Prático:
    Criação de um formulário HTML simples (campos como nome, e-mail, etc.).
    Revisão de conceitos de CSS:
    Estilização de formulários (inputs, labels e botões).
    Trabalhando com cores, fontes, paddings e margins.
    Utilização de Flexbox para alinhamento e centralização.
    Tornando o layout responsivo.
    Utilização de JavaScript para capturar o evento de submit do formulário.
    Validação básica dos campos preenchidos.
    Armazenamento dos dados no navegador utilizando localStorage.setItem().



Tecnologias Utilizadas:
    HTML5
    CSS3
    JavaScript

Como testar:
    Abra o arquivo HTML em qualquer navegador moderno.
    Preencha o formulário (a estilização deve facilitar a experiência do usuário).
    Ao enviar, os dados serão salvos automaticamente no seu navegador.
    Abra as Ferramentas do Desenvolvedor (F12) > Aplicação (Application) > Armazenamento Local (Local Storage) para visualizar os dados salvos.
